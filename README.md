## 🚀 How to Run

1.Make sure Rust & Cargo is installed 🦀
   ```sh
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh](https://rustup.rs/)
   ```
2. Clone this repositori
   ```sh
   (https://github.com/FaizNation/myRust.git)
   ```
3. Jalankan contoh kode
   ```sh
   cargo run
   ```
